# 📄 法院文书下载助手
Chrome 浏览器插件，帮助当事人一键下载属于自己的法律文书。

## ⚠️ 重要声明
本插件仅适用于：
- ✅ 案件当事人本人
- ✅ 持有授权的法律代理人
- ❌ 严禁用于下载他人文书或商业用途

## 功能
- 自动识别当前页面的案件信息
- 显示所有可下载文书列表
- 一键批量下载全部文书

## 安装方法

### 方法一：手动安装（推荐）
1. 点击右上角绿色 Code 按钮 → Download ZIP
2. 解压到任意文件夹
3. 打开 Chrome → 地址栏输入 chrome://extensions/
4. 右上角开启 "开发者模式"
5. 点击 "加载已解压的扩展程序"
6. 选择解压后的文件夹

## 使用方法
1. 打开法院网站并登录
   https://zxfw.court.gov.cn
2. 进入你的案件文书页面
3. 点击浏览器右上角插件图标
4. 点击 "一键下载全部"

## 常见问题
**Q：为什么显示获取失败？**
A：请确保已登录法院网站，并刷新页面后重试

**Q：下载的文件在哪里？**
A：在浏览器默认下载文件夹内
