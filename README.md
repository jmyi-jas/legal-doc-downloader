#  法院文书下载助手
浏览器插件，Chrome等浏览器都能用，帮助当事人一键下载属于自己的法律文书。

## 功能
- 自动识别当前页面的案件信息
- 显示所有可下载文书列表
- 一键批量下载全部文书

## 安装方法

### 手动安装（推荐）
1. 点击右上角绿色 Code 按钮 → Download ZIP
2. 解压到任意文件夹
3. 打开 Chrome → 地址栏输入 chrome://extensions/【或者如果你是edge浏览器，则输入：edge://extensions/，本质上没有区别，下面的步骤也一样】
4. 右上角开启 "开发者模式"
5. 点击 "加载已解压的扩展"
6. 选择解压后的文件夹

## 使用方法
1. 打开法院网站并登录
   https://zxfw.court.gov.cn
2. 进入你的案件文书页面【上面两个步骤也可以直接替换成点击短信所附的链接，显示文书列表的界面即可】
3. 点击浏览器右上角插件图标
4. 点击 "一键下载全部"

## 常见问题
**Q：为什么显示获取失败？**
A：请确保已登录法院网站，并刷新页面后重试

**Q：下载的文件在哪里？**
A：在浏览器默认下载文件夹内
